Texly Chat Widget Test Page

This is a test page for the Texly Chat widget. The chat widget should appear in the bottom-right corner.

Testing Instructions:

  1. Make sure your server is running on http://localhost:8080
  2. Replace YOUR_BOT_ID in the script tag below with an actual bot ID
  3. The widget launcher button should appear in the bottom-right corner
  4. Click it to open the chat window
  5. Type a message and verify SSE streaming works
